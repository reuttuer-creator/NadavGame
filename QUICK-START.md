# 🚀 Quick Start - PWA Setup

## צעדים מהירים:

### 1️⃣ צור אייקונים (נפתח אוטומטית):
- הדף `create-icons.html` נפתח בדפדפן
- לחץ "Download All Icons" 
- שמור את 8 הקבצים בתיקיית הפרויקט

### 2️⃣ העלה ל-Git:
```bash
# הוסף את האייקונים שהורדת
git add icon-*.png

# הוסף את שאר הקבצים
git add .

# צור commit
git commit -m "Added PWA support v1.3.0 - installable mobile app"

# העלה
git push origin main
```

### 3️⃣ המתן ל-Vercel:
- Vercel יעדכן אוטומטית תוך 1-2 דקות
- בדוק ב-Vercel Dashboard

### 4️⃣ התקן על הטלפון:
**Android:** תפריט → "התקן אפליקציה"  
**iOS:** שיתוף → "Add to Home Screen"

---

## ✅ מה נוסף?

- 📴 משחק אופליין
- 🚀 טעינה מהירה (Service Worker)
